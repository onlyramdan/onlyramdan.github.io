👨‍💻 About Me

Seorang lulusan Teknik Rekayasa Komputer yang memiliki keahlian mendalam di bidang Website Development, dan Internet of Thing. Ahli dalam bahasa pemrograman Go, Ruby, PHP, JavaScript, C#, dan Python. Selain itu, memiliki pengalaman menggunakan framework seperti Laravel, CodeIgniter, Gin, .Net, dan Ruby on Rails, juga berpengalaman menggunakan MySQL, PostgreSQL, dan MangoDB untuk manajemen data. Terampil dalam komunikasi, kerja sama, dan manajemen waktu. Berkomitmen untuk terus belajar demi meningkatkan kemampuan diri serta dapat berkontribusi bagi kemajuan perusahaan.

💻 Project

Web LSP HCMI

Web Arsip LSP HCMI

IoT Monitoring Lingkungan

Merupakan sistem yang memonitoring lingkungan di perpustakaan IPB University dengan indikator suhu, kelembapan, dan CO2 menggunakan komunikasi MQTT. Sistem ini berfungsi menjaga kenyamanan pengunjung dan staf.

Website LegalinAja

Pakan Otomatis di Tazkia Farm

Project ini bertujuan membuat sebuah pakan otomatis untuk burung puyuh penghasil telur agar sesuai dengan takarannya. Dalam project ini menggunakan mikrokontroler arduino dan sensor loadcell hx711 untuk mengatur berat pakan yang akan dikeluarkan.

IoT aFEESH

Project ini membuat sebuah pakan otomatis dengan menggunakan sistem serta berbasis Web dimana pakan akan keluar otomatis ketika pakan waktunya sudah tepat sama seperti yang di setting di pada website dan juga berdasarkan kualitas air kolam lele.

Angsuran VA ULLAM
